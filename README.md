# MaroonSchedules

**Quick Start**:<br>
Run the following:<br>
git clone https://github.com/rianp0071/MaroonSchedules<br>
cd MaroonSchedules<br>
npm install expo<br>
npx expo start<br>

**Languages/Frameworks:**

**Frontend:** React Native, Typescript, HTML, CSS

**Backend:** Python, FastApi

**Database:** SQL
