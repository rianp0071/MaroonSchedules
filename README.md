# MaroonSchedules

<ins>**Quick Start Demo (run the following)**:</ins><br>
git clone https://github.com/rianp0071/MaroonSchedules<br>
cd MaroonSchedules<br>
cd Backend<br>
pip install -r requirements.txt<br>
***Make .env file in root project directory and paste "EXPO_PUBLIC_API_URL=http://your-ipv4-address-here:8000" into it***<br>
uvicorn main:app --reload --host 0.0.0.0<br>

***Make new terminal***<br>
***cd to MaroonSchedules if not already***<br>
npm install expo<br>
npx expo start<br>

<ins>**Languages/Frameworks:**</ins>

**Frontend:** React Native, Typescript, HTML, CSS

**Backend:** Python, FastApi

**Database:** SQL
